# Fake News Detection System

This project implements a fake news detection model using classical machine learning techniques.

## Overview
The model classifies news articles as fake or real based on their textual content.

## Approach
- Text feature extraction using TF-IDF
- Classification using Logistic Regression

## Dataset
Fake and Real News Dataset (Kaggle)
